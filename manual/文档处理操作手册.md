## 二、合同审核

合同审核功能支持对各类商务合同进行智能分析，识别潜在风险点，审查条款合规性，并生成专业的审核报告。

### 操作流程

1. **上传合同文件**：将合同文档（PDF、Word、图片等）上传到用户目录
2. **输入审核指令**：对合同进行审核分析
3. **查看审核结果**：系统自动识别风险并生成报告

上传合同到目录中
![image.png](https://pookingcn.myds.me:38090/v/2603/69b263cb64b71.png)

用户输入
```
查看我的房屋租赁的合同，查看合同内容并给出审核意见
```

![image.png](https://pookingcn.myds.me:38090/v/2603/69b266b91e192.png)

查看在目录中的详细报告
![image.png](https://pookingcn.myds.me:38090/v/2603/69b2672081bda.png)

### 审核报告内容

合同审核完成后，系统生成详细的审核报告，包含以下内容：

- **总体评估**：合同整体风险等级（高/中/低）
- **问题清单**：发现的问题及风险点列表
- **修改建议**：针对问题的具体修改建议
- **补充条款**：建议补充的条款内容
- **审核意见**：综合审核结论

### 风险等级说明

|风险等级|标识|说明|
|---|---|---|
|高风险|🔴|存在重大法律风险，建议修改后使用|
|中风险|🟡|存在一定风险，建议完善条款|
|低风险|🟢|条款基本合规，可正常使用|

### 常见合同风险点

- 条款表述模糊，易产生歧义
- 双方权利义务不对等
- 违约责任约定不明确
- 缺少争议解决条款
- 付款条款不清晰
- 保密条款缺失或不完善
- 知识产权归属不明确
- 不可抗力条款缺失

### 导出审核报告

审核完成后，可将报告导出或发送至邮箱：

```
把合同审核报告发送到邮箱：XXX@ospreyai.cn
```

## 四、文档格式转换

文档格式转换功能支持多种常用文档格式之间的相互转换，方便文档处理和分享。

### 支持的转换格式

| 源格式 | 目标格式 | 说明 |
|--------|----------|------|
| Word (.docx) | Markdown (.md) | 将 Word 文档转换为 Markdown 格式 |
| Markdown (.md) | Word (.docx) | 将 Markdown 转换为 Word 文档 |
| Word (.docx) | PDF (.pdf) | 将 Word 文档转换为 PDF 格式 |
| Markdown (.md) | PDF (.pdf) | 将 Markdown 转换为 PDF 格式 |

### 操作说明

#### 转成 PDF 格式

将文档转换为 PDF 格式，便于分享和打印。

**操作步骤：**
1. 将需要转换的文件上传到用户目录
2. 输入转换指令，例如：
```
把目录中的 xxx.docx 转成 pdf
```
或
```
把目录中的 xxx.md 转成 pdf
```
3. 系统自动转换并生成 PDF 文件

![image-20260316110105260](https://pookingcn.myds.me:38090/v/2603/69b7726fc88ae.png)

![image-20260316110537751](https://pookingcn.myds.me:38090/v/2603/69b7738036105.png)

#### 转成 MD 格式

将其他格式的文档转换为 Markdown 格式，便于编辑和版本控制。

**操作步骤：**
1. 将需要转换的文件上传到用户目录
2. 输入转换指令，例如：
```
把目录中的 xxx.docx 转成 md
```
3. 系统自动转换并生成 Markdown 文件

## 五、常用指令模板

本节汇总了文档处理相关的常用指令模板，方便快速复制使用。

### 5.1 发票提取类

```
# 提取发票信息
提取发票中的信息

# 提取指定发票
提取目录下 XX 发票的信息
```

### 5.2 合同审核类

```
# 审核合同
查看我的XX合同，审核下内容

# 查看详细报告
查看合同审核报告

# 发送报告到邮箱
把合同审核报告发送到邮箱：XXX@ospreyai.cn
```

### 5.3 邮件发送类

```
# 发送审核报告
把审核报告发送到邮箱：XXX@ospreyai.cn

# 发送图片到邮箱
把目录中的图片发送到邮箱：XXX@ospreyai.cn

# 发送指定文件
把 XX 文件发送到邮箱：XXX@ospreyai.cn

# 修复邮件附件问题
收到邮件的附件文件的名称为空，请检查后重新发送
```

### 5.4 文档格式转换类

```
# Word 转 PDF
把目录中的 xxx.docx 转成 pdf

# Markdown 转 PDF

# Word 转 Markdown
```
