# 专业客服知识库操作手册 v1.0.0

> 本手册介绍专业客服知识库的核心操作流程，涵盖查看、上传和使用三个场景，适用于需要通过知识库提供智能问答服务的客服人员。

本节介绍如何在系统中查看已有的知识库内容。

### 1.1 操作步骤

**步骤 1：输入查看指令**

用户在系统输入框中输入查看知识库的相关指令。

![用户输入查看指令](https://pookingcn.myds.me:38090/v/2603/69c1df1da2fb1.png)

**步骤 2：等待技能自动加载**

系统识别指令后，自动加载知识库相关技能。

![技能自动加载](https://pookingcn.myds.me:38090/v/2603/69c1df630c43b.png)

**步骤 3：查看知识库内容**

技能加载完成后，系统显示知识库的查询结果。

![显示知识库结果](https://pookingcn.myds.me:38090/v/2603/69c1df8ad08dd.png)

## 二、上传知识库

本节介绍如何向知识库上传新的文档内容。

### 2.1 操作步骤

**步骤 1：打开知识库问答目录**

在用户文件目录中，找到并打开「知识库问答」文件夹。

![打开知识库问答目录](https://pookingcn.myds.me:38090/v/2603/69c1e157e7bd5.png)

**步骤 2：上传文件**

将需要添加为知识库内容的文件上传到该目录中。

![上传文件到目录](https://pookingcn.myds.me:38090/v/2603/69c1e1dfba3af.png)

**步骤 3：等待系统解析**

文件上传后，系统会自动将文件上传到知识库，并进行解析和向量存储。

> **注意**：根据文件数量和大小的不同，解析所需时间会有所差异，通常约 **5 分钟左右** 即可使用。

![文件上传成功](https://pookingcn.myds.me:38090/v/2603/69c1e29670bc6.png)

### 2.2 目录创建说明

如果在用户目录下没有找到「知识库问答」文件夹，可以手动创建一个同名文件夹。

## 四、注意事项

- **解析时间**：知识库文件上传后，根据文件数量和大小的不同，等待上传、解析的时间会有所差异。请耐心等待系统完成解析后再使用。

- **目录缺失处理**：如果用户目录下没有找到「知识库问答」的文件夹，可以手动创建一个。

- **解析异常处理**：若知识文件已上传但 `chunk` 数为 0，可以手动使用以下指令尝试解决：

  ```
  把知识库中的XX文件解析一下
  ```

- **优先使用知识库**：若需要系统严格按知识库内容输出结果，建议用户明确说明以下内容：

  ```
  以后问题的询问，优先使用知识库中的结果；或没有匹配或匹配值过低再尝试其他信息源。把这个准则记录并保存下来
  ```

## 五、常见问题

### 5.1 为什么上传文件后无法立即使用？

文件上传后需要经过解析和向量存储过程，通常需要约 5 分钟。请确认文件已成功上传并等待足够时间后再尝试查询。

### 5.2 如何确认文件已成功解析？

可以通过查看知识库内容确认文件是否已成功解析。若查询时能返回相关内容，说明解析成功。

### 5.3 上传的文件有什么格式要求？

系统支持常见的文档格式。具体支持的文件格式请参考系统相关说明文档。
