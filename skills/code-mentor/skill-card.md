## Description: <br>
Comprehensive AI programming tutor for all levels that teaches through interactive lessons, code review, debugging guidance, algorithm practice, project mentoring, and design pattern exploration, with support focused on Python and JavaScript. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>

## Use Case: <br>
Developers, students, and interview candidates use this skill to learn programming concepts, debug code, review implementations, practice algorithms and data structures, and plan software projects with guided feedback. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The optional test helper may run local test suites and execute project code on the user's machine. <br>
Mitigation: Use it only on trusted repositories, avoid projects containing secrets, and review the target code and test command before execution. <br>
Risk: The skill stores learning progress locally in a learning log. <br>
Mitigation: Review, edit, or remove the learning log when progress notes should not be retained. <br>
Risk: Programming guidance, reviews, and debugging suggestions may be incomplete or incorrect. <br>
Mitigation: Validate suggested changes with tests, code review, and project-specific requirements before relying on them. <br>
