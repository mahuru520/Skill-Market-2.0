## Known Risks and Mitigations: <br>
Risk: The skill includes executable examples for installing a third-party CLI. <br>
Mitigation: Review SKILL.md before installing and prefer manual download with checksum verification over piping the installer to a shell. <br>
Risk: Drafts, unpublished posts, or private code could be sent to external services through inference.sh examples. <br>
Mitigation: Use an inference.sh account with appropriate limits and avoid sending confidential drafts or private code to external services. <br>
Risk: The X posting example can publish text publicly. <br>
Mitigation: Review the final text and run the posting example only when you explicitly intend to publish it. <br>
