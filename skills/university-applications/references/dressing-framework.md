# 穿衣 / 穿搭建议框架 (Dressing / Styling Framework)

> ⚠️ **定位说明**
> - 本 skill 的穿衣建议基于**传统五行色彩理论**与**个人命理气场分析**，融合现代穿搭美学。
> - 不替代专业形象顾问、时尚设计师或色彩诊断师的意见。
> - 穿搭建议仅作文化参考与个人风格探索，不构成消费引导或品牌推荐。

## 核心框架

### 1. 五行色彩体系

#### 基础五行色

| 五行 | 基础色 | 衍生色 | 能量特质 |
|------|--------|--------|---------|
| 木 | 绿、青 | 翠、碧、草绿、森林绿 | 生发、舒展、清新 |
| 火 | 红、紫 | 橙、玫红、酒红、珊瑚 | 热情、醒目、活跃 |
| 土 | 黄、棕 | 卡其、驼色、米色、咖啡 | 稳重、包容、踏实 |
| 金 | 白、银 | 灰、白金、香槟、金属色 | 利落、收敛、精致 |
| 水 | 黑、蓝 | 藏青、靛蓝、深灰、炭黑 | 深沉、流动、智慧 |

#### 色彩运用原则

- **主色**：占全身 60% 左右（外套、裤装/裙装、大面积单品）
- **辅色**：占 30% 左右（内搭、衬衫、配饰）
- **点缀色**：占 10% 左右（围巾、包包、首饰、鞋子）

#### 喜用神色彩应用

- 喜用神为木 → 绿色系为主/辅色，点缀蓝/黑（水生木）
- 喜用神为火 → 红色系为主/辅色，点缀绿/青（木生火）
- 喜用神为土 → 黄色/棕色系为主/辅色，点缀红/紫（火生土）
- 喜用神为金 → 白色/银色系为主/辅色，点缀黄/棕（土生金）
- 喜用神为水 → 黑色/蓝色系为主/辅色，点缀白/银（金生水）

> **注意**：色彩运用不追求全身一色，讲究"主辅点"层次。不应要求用户为了五行而强行穿不适合肤色或场合的颜色。

### 2. 八字气场与风格方向

| 日主天干 | 气质倾向 | 适合风格 |
|---------|---------|---------|
| 甲木 | 挺拔、正直、有主见 | 简洁利落、结构感强、风衣/西装 |
| 乙木 | 柔韧、细腻、有审美 | 柔和线条、文艺、蕾丝/飘逸感 |
| 丙火 | 热情、外放、有感染力 | 明亮色彩、设计感、重视配饰 |
| 丁火 | 内敛、精致、有温度 | 细节品质、低调奢华、小而精 |
| 戊土 | 厚重、可靠、包容 | 大地色系、经典款、质感面料 |
| 己土 | 温润、随和、细腻 | 柔和中性色、舒适休闲、棉麻 |
| 庚金 | 刚健、果断、有棱角 | 黑白灰、极简、金属配饰 |
| 辛金 | 精致、敏锐、有品位 | 高级感、珠宝点缀、丝绸/羊绒 |
| 壬水 | 流动、开阔、有智慧 | 深蓝/黑色系、宽松剪裁、层次感 |
| 癸水 | 细腻、深沉、有洞察 | 暗色系、神秘气质、精致细节 |

> 无八字时，此部分略过，专注通用穿搭原则。

### 3. 场合适配

| 场合 | 核心原则 | 色彩建议 |
|------|---------|---------|
| 面试/正式场合 | 稳重、专业、收敛 | 深蓝、灰、黑、白、卡其 |
| 社交/聚会 | 有亮点但不张扬 | 辅色提亮，点缀色出彩 |
| 约会/感情 | 柔和、亲和、有温度 | 暖色辅色，柔和过渡 |
| 重要谈判 | 气场、自信、掌控感 | 深色系主色，金属点缀 |
| 休养生息 | 舒适、放松、回归自然 | 大地色、柔和绿、米白 |
| 创意/表达 | 个性、有记忆点 | 可大胆用喜用色做主色 |

### 4. 季节与流年色彩

- **春**：木旺，绿色系当令；喜木者顺势，忌木者收敛
- **夏**：火旺，红色系当令；喜火者顺势，忌火者清凉
- **秋**：金旺，白色/金属色当令；喜金者顺势，忌金者暖色平衡
- **冬**：水旺，黑色/深蓝当令；喜水者顺势，忌水者暖色调和

**流年参考**：可结合当年天干地支的五行旺衰，给流年穿搭色彩倾向（如甲辰年木土旺，绿色、黄色系较活跃）。

### 5. 肤色适配（通用原则，不依赖八字）

- **暖肤色**（偏黄/偏金）：适合暖色系（橙、黄、暖红、卡其、咖啡）
- **冷肤色**（偏粉/偏蓝）：适合冷色系（蓝、紫、冷粉、银灰、藏青）
- **中性肤色**：冷暖皆可，自由度高

> 肤色适配优先于五行色彩。若喜用色与肤色不适配，建议将喜用色用于"辅色"或"点缀色"层级。

## 输出结构

### 标准版（S/A 级）

1. **五行色彩方向**：喜用神对应色系 + 生助色系
2. **日常穿搭色谱**：推荐 3-5 个主色 + 辅色 + 点缀色组合
3. **风格方向**：基于日主/命格气质的气质判断
4. **场合建议**：针对用户提到的具体场合给搭配思路
5. **当季/流年色彩提示**：当前季节或流年的色彩能量
6. **避雷提醒**：明显不适合的色彩或风格（非强制，仅参考）
7. **点醒句**：收尾

### 轻量版（B/C 级）

1. **通用色彩搭配**：基于季节/场合的基础建议
2. **五行色彩参考**：如有出生信息可粗略判断
3. **一句核心建议**：最关键的穿衣方向

## 硬性边界

| 禁止行为 | 原因 |
|---------|------|
| 声称穿衣可以"改运""招财""招桃花" | 禁止夸大与商业欺诈 |
| 推荐具体品牌或商品链接 | 不做商业引导 |
| 要求用户购买特定衣物 | 不替代消费决策 |
| 断言不穿某色会倒霉 | 禁止恐吓式营销 |
| 替代形象顾问做专业诊断 | 仅作文化参考 |
| 忽视用户实际条件（肤色/体型/预算） | 保持现实合理性 |
