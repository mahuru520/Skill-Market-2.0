# 面相 / 观人解读框架 (Physiognomy Framework)

> ⚠️ **隐私与安全提示**
> - 人脸识别数据属于**高敏感生物识别信息**，泄露后不可撤销，可能用于身份识别、跟踪与深度伪造。
> - 本 skill **不自动保存**面部照片到本地档案；请仅在当次会话中使用。
> - 强烈建议以**文字描述**替代原始照片（如"额头宽、眉清目秀、颧骨稍高"）。
> - 请勿为未成年人、他人、公众人物提交面部照片进行面相解读。
> - 如仅出于兴趣尝试，可先描述面部特征文字，再决定是否需要发图。

## 核心框架

### 1. 三庭（纵向格局）

| 区域 | 范围 | 主事 | 理想状态 |
|------|------|------|---------|
| 上庭 | 发际线至眉心 | 少年运（15-30）、先天智力、祖荫 | 开阔饱满 |
| 中庭 | 眉心至鼻尖 | 中年运（31-50）、事业心、执行力 | 挺拔端正 |
| 下庭 | 鼻尖至下巴 | 晚年运（51+）、晚福、稳定性 | 圆润有肉 |

**三庭比例**：均衡者为佳。某庭偏长/偏短 → 对应人生阶段节奏差异。

### 2. 五眼（横向格局）

面部宽度以五个眼距为标准。两眼之间过宽/过窄、眼距不均 → 性格与社交模式差异。

### 3. 十二宫

| 宫位 | 位置 | 主事 |
|------|------|------|
| 命宫 | 两眉之间（印堂） | 整体运势、气度、福祸根基 |
| 财帛宫 | 鼻子 | 资源管理与财富观 |
| 兄弟宫 | 眉毛 | 手足关系、交友模式 |
| 田宅宫 | 眉眼之间（上眼睑） | 不动产、居家、安全感 |
| 男女宫 | 眼下（卧蚕/子女宫） | 子女缘、感情表达 |
| 奴仆宫 | 下巴两侧（腮部） | 下属/晚辈关系、社交支持 |
| 妻妾宫 | 眼尾外侧（奸门） | 感情/婚姻状况（仅用于成人） |
| 疾厄宫 | 鼻梁中段 | 健康体质参考（不替代医疗） |
| 迁移宫 | 额角两侧（天仓） | 外出/变动/环境适应 |
| 官禄宫 | 额头正中 | 学业/事业/社会地位 |
| 福德宫 | 眉尾上方 | 精神世界、满足感、贵人缘 |
| 父母宫 | 额角偏上（日月角） | 父母缘、长辈关系 |

### 4. 五官精要

#### 眉
- **浓淡**：浓眉 → 重情义、行动力；淡眉 → 理性、偏冷静
- **长短**：长眉 → 有耐力；短眉 → 急性子
- **弯直**：弯眉 → 圆融；直眉 → 刚直
- **眉间距**：宽 → 开朗；窄 → 思虑深

#### 眼
- **大小**：大眼 → 外向、易感；小眼 → 内敛、观察力强
- **眼神**：清亮 → 心气足；浑浊 → 疲惫/困惑期
- **眼尾**：上挑 → 进取；下垂 → 温和/感性

#### 鼻
- **高挺**：自尊心强、主见明确
- **圆润**：包容、务实
- **鼻翼**：丰隆 → 理财保守稳健；薄 → 花钱决断快

#### 嘴
- **大小**：大嘴 → 表达欲强；小嘴 → 谨慎内敛
- **唇厚**：重情、感性；唇薄 → 理性、重逻辑
- **嘴角**：上扬 → 乐观；下垂 → 偏严肃/谨慎

#### 耳
- **大小**：大耳 → 精力旺；小耳 → 心思细
- **贴脑/招风**：贴脑 → 稳；招风 → 活跃
- **耳垂**：厚大 → 传统福相说法；薄 → 偏敏感

#### 下巴
- **方圆**：方下巴 → 意志力强；圆下巴 → 圆融、人缘好
- **尖削**：敏感、变动性大

## 气色与近期状态

面相解读可加入气色判断作为"近期状态快照"：

| 区域 | 气色 | 倾向 |
|------|------|------|
| 印堂 | 明亮 → 神清气爽；发暗 → 压力/困顿期 |
| 眼眶 | 青黑 → 睡眠不足/思虑重 |
| 颧骨 | 红润 → 精力旺；苍白 → 气虚 |
| 唇色 | 红润 → 气血足；发紫/白 → 需关注健康 |

> 气色判断仅作生活作息参考，**不替代医疗诊断**。

## 输出结构

### 标准版

1. **三庭总论**：整体格局与人生阶段节奏
2. **十二宫精讲**：命宫、财帛、事业、感情等重点宫位
3. **五官分析**：眉眼鼻嘴耳逐一看性格与行事风格
4. **气色快照**：近期身心状态判断
5. **综合印象**：整体气质与气场描述
6. **自我观察方向**：可留意的性格盲区或优势
7. **点醒句**：收尾

### 轻量版（B/C 级或文字描述时）

1. **三庭一句话**：上中下庭整体印象
2. **性格倾向**：五官组合的气质判断
3. **近期状态**：气色/神态反映的状态
4. **一句提醒**：仅作文化参考，健康疑虑请就医

## 硬性边界

| 禁止行为 | 原因 |
|---------|------|
| 声称通过面相下医疗诊断 | 不替代医学检查 |
| 断言寿命或重大疾病 | 禁止恐吓式预测 |
| 声称面相可以"破解""改运" | 禁止商业欺诈 |
| 给未成年人下宿命标签 | 禁止命定化表达 |
| 替他人面相下负面定性 | 除非对方知情同意 |
| 面相与职业/投资直接挂钩 | 不替代专业建议 |
| 基于种族/面部特征的歧视判断 | 禁止偏见输出 |
