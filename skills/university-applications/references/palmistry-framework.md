# 掌纹 / 手相解读框架 (Palmistry Framework)

> ⚠️ **隐私与安全提示**
> - 掌纹属于**生物识别特征信息**，一旦泄露不可撤销。
> - 本 skill **不自动保存**手掌照片到本地档案；请仅在当次会话中使用。
> - 如需长期参考，建议以**文字描述**（如"生命线偏长、感情线分叉"）替代原始图片。
> - 不要上传包含明显身份标识（如手背纹身、戒指特写、背景环境）的原图。

---

## 适用场景

| 场景 | 示例 |
|------|------|
| 性格与先天倾向 | "我的性格有什么特点" |
| 健康状态参考 | "身体有没有什么需要注意的地方" |
| 情感模式 | "我在感情里容易有什么问题" |
| 事业与发展轨迹 | "事业线怎么看" |
| 当下状态快照 | "最近压力大，帮我看看手掌反映什么" |

---

## 资料完整度分级

| 级别 | 条件 | 处理方式 |
|------|------|---------|
| **S 级** | 双手高清照片 + 光线清晰 + 手型完整 | 逐线精读，左右手对比 |
| **A 级** | 单手清晰照片 + 掌纹可见 | 标准版解读，聚焦主线 |
| **B 级** | 图片模糊 / 仅有文字描述 | 轻量版，讲趋势与模式 |
| **C 级** | 仅口头提问无照片无描述 | 推荐其他体系（塔罗/八字） |

## 主线解读顺序

### 1. 手型总论（先看整体）

| 手型 | 特征 | 倾向 |
|------|------|------|
| 火型手 | 手掌方、手指短 | 热情、冲动、执行力强 |
| 土型手 | 手掌厚、手指粗 | 务实、稳定、耐力强 |
| 风型手 | 手掌方、手指长 | 理性、沟通、好奇心强 |
| 水型手 | 手掌长、手指 slender | 敏感、直觉、情绪化 |

### 2. 三大主线

#### 生命线（围绕拇指的弧线）
- **深长清晰**：体质基础好，恢复力强
- **浅淡断续**：精力波动大，需注意休息节奏
- **分叉**：人生重大转折或生活方式改变
- **链形纹**：阶段性健康波动（不替代医疗诊断）

#### 智慧线（横贯手掌中部）
- **长而直**：逻辑性强，理性决策
- **短而弯**：直觉型，依赖经验与感觉
- **分叉**：多元兴趣，思维灵活
- **断裂**：思维模式转变期

#### 感情线（小指下方横线）
- **长至食指**：理想主义者，感情投入深
- **长至中指**：务实型感情观
- **短**：情感表达含蓄
- **分叉**：感情选择期或关系模式变化
- **岛纹**：情感困惑期（当前或过往）

### 3. 重要辅线

| 线名 | 位置 | 主事 |
|------|------|------|
| 命运线 / 事业线 | 手腕至中指 | 事业发展、人生方向感 |
| 太阳线 | 无名指下竖线 | 创造力、名声、艺术天赋 |
| 婚姻线 | 小指下方横纹 | 感情契约、重要关系 |
| 健康线 | 小指与生命线间 | 亚健康提示（不替代医疗） |
| 财运线 | 小指与无名指间 | 资源管理能力 |

### 4. 丘位观察

| 丘位 | 位置 | 主事 |
|------|------|------|
| 金星丘（拇指根部） | 生命力、活力 |
| 火星丘（拇指两侧） | 行动力 / 忍耐力 |
| 土星丘（中指下） | 责任感、自律 |
| 太阳丘（无名指下） | 创造力、社交 |
| 水星丘（小指下） | 沟通、商业敏感 |
| 月亮丘（手腕外侧） | 直觉、想象力 |

## 左右手原则

- **惯用手**（通常右手）：后天发展、当前状态、正在形成的模式
- **非惯用手**（通常左手）：先天禀赋、遗传底色、原生家庭影响
- **解读时**：先看先天（左手）→ 再看后天（右手）→ 对比变化

---

## 输出结构

### 标准版

1. **手型总论**：整体性格底色
2. **主线解读**：生命 / 智慧 / 感情逐条分析
3. **辅线与丘位**：事业、财运、健康提示
4. **左右对比**：先天与后天的变化
5. **当前状态快照**：手掌反映的近期身心状态
6. **自我观察方向**：不替代专业判断，给反思角度
7. **点醒句**：收尾

### 轻量版（B/C 级或文字描述时）

1. **主线一句话**：三主线整体印象
2. **性格倾向**：基于描述的判断
3. **近期状态**：压力 / 变化 / 调整期
4. **一句提醒**：仅作文化参考，健康疑虑请就医

## 硬性边界

| 禁止行为 | 原因 |
|---------|------|
| 声称通过掌纹下医疗诊断 | 不替代医学检查 |
| 断言寿命或重大疾病 | 禁止恐吓式预测 |
| 声称掌纹可以"破解""改运" | 禁止商业欺诈 |
| 对未成年人下宿命标签 | 禁止命定化表达 |
| 掌纹与职业/投资直接挂钩 | 不替代专业建议 |
