## 各体系最少需要什么资料

### 八字 / 四柱
**最理想**：公历出生年月日 + 出生时间（精确到小时）+ 出生地/城市 + 想问的问题

**没有时辰时**：只能做半精度解读，不把时柱说死

### 紫微斗数
**最理想**：出生年月日 + 出生时间 + 出生地 + 性别 + 或直接发命盘截图

### 塔罗
**方式 A（用户自己抽牌）**：牌阵名称 + 每张牌 + 正位/逆位 + 问题背景

**方式 B（象征性抽牌）**：主题 + 当前困惑 + 时间范围 → 说明是反思式/投射式阅读

### 西方星盘
**最理想**：出生年月日 + 出生时间 + 出生地 + 想看主题

**没有时间时**：重点看太阳、月亮、行星主题，不把上升和精细宫位说成定论

### 数字命理
**至少需要**：出生年月日

**若加入姓名分析**：用户确认姓名写法（中文名按拼音还是固定英文名）

### 奇门遁甲
**最理想**：明确问题 + 起局时间/盘面截图 + 事项背景

**没有盘时**：可做"奇门导向"的阶段判断，不假装具体门星神都已落准

### 六爻 / 易经卦象
**最理想**：卦名 + 动爻 + 世应/用神信息 + 起卦时间 + 问题背景

**只有一个卦象结论时**：做主线判断，不装完整纳甲细盘

### 梅花易数
**可用**：起念时间 / 数字 / 时间方向触发事件 / 用户当下最强的问题

### 九宫飞星 / 风水
**至少需要**：空间类型 + 基本布局描述 + 朝向或门窗位置 + 想改善的重点

**飞星计算**：基于洛书九宫与流年干支进行理论推演，提供空间方位参考框架（不涉及外部数据调用）

### 掌纹 / 手相
**推荐方式**：以文字描述掌纹特征（生命线/智慧线/感情线/事业线的长度、深浅、分叉、岛纹等）作为首选输入，安全且无泄露风险。

**照片替代**：仅在用户主动要求且充分知情的情况下，可接受当次会话中的双手照片（自然光、无遮挡）。照片不会被保存到本地档案，会话结束后即丢弃。

> ⚠️ **隐私提醒**：掌纹属于不可撤销的生物识别信息。强烈建议优先使用文字描述；如发送照片，请避免背景中出现可识别身份的环境特征（如房间布局、地标、手背纹身等）。本 skill 不自动保存任何图片。

### 面相 / 观人
**推荐方式**：以文字描述面部特征（三庭比例、眉型、眼型、鼻型、唇型、下巴形状、气色等）作为首选输入，安全且无泄露风险。

**照片替代**：仅在用户本人主动要求、充分知情且照片仅涉及自己时，可接受当次会话中的素颜照片。照片不会被保存到本地档案，会话结束后即丢弃。严禁为未成年人、他人或公众人物提交面部照片。

> ⚠️ **隐私提醒**：人脸属于高敏感生物识别数据，一旦泄露不可撤销，可能被用于身份识别、跟踪或深度伪造。强烈建议优先使用文字描述；如发送照片，请避免背景中出现可识别身份的环境特征。

### 合盘 / 关系分析
**至少需要**：双方关系背景 + 双方出生资料 + 当前阶段 + 最想看的问题

---

### 起名 / 命名
**最理想**：出生年月日时 + 性别 + 姓氏 + 家族辈分字（如有）+ 父母期望/偏好

**最低要求**：姓氏 + 性别

**额外信息**：如果用户已有一个候选名字，可直接做名字分析

> 起名涉及法律身份与社会交往，建议用户结合家庭意愿、实际使用便利性综合决定。本 skill 不做最终命名决定。

### 穿衣 / 搭配
**最理想**：出生年月日时 + 性别 + 体型描述 + 日常风格偏好 + 具体场合

**最低要求**：性别 / 季节 / 场合

> 穿衣建议基于五行色彩理论与现代美学，不替代专业形象顾问。实际穿搭应结合肤色、体型、预算、场合等现实条件。

## 分级路由

| 级别 | 条件 | 处理 |
|------|------|------|
| **S 级** | 盘面/牌阵/卦盘/户型图已给 | 进入精读，多角度深讲 |
| **A 级** | 生日时地/起局信息/布局信息完整 | 进入标准版，提醒误差 |
| **B 级** | 资料不完整 | 进入轻量版，聚焦趋势 |
| **C 级** | 只有问题没有资料 | 推荐塔罗/梅花/综合式解读 |

- "你这次我先按轻量趋势给你看主线。"
- "没给精确资料，我不做假精排盘，先讲更稳的部分。"
- "这一版更适合看你当前状态、关系动力和下一步重点。"
