## 一、功能角色矩阵说明

### 1.1 功能描述

功能名称：用户管理功能设计

所属模块：账号与权限

功能描述：用于维护平台账号基础信息、角色授权关系和账号启停状态，支撑后台用户的统一管理。

### 1.2 角色权限总览

| 角色 | 功能权限 | 数据权限 |
| --- | --- | --- |
| 系统管理员 | 查看、创建、编辑、删除、导入、导出 | 全量数据 |
| 业务管理员 | 查看、创建、编辑、导出 | 所属组织数据 |
| 审核人员 | 查看、审核 | 待审核数据 |
| 普通用户 | 查看本人资料 | 本人数据 |

### 1.3 功能角色矩阵

| 功能点 | 系统管理员 | 业务管理员 | 审核人员 | 普通用户 |
| --- | --- | --- | --- | --- |
| 查看列表 | 是 | 是 | 是 | 否 |
| 查看详情 | 是 | 是 | 是 | 是 |
| 新增 | 是 | 是 | 否 | 否 |
| 编辑 | 是 | 是 | 否 | 否 |
| 删除 | 是 | 否 | 否 | 否 |
| 导出 | 是 | 是 | 否 | 否 |

### 3.2 信息查询区

| 重置 | 按钮 | 居右 | 可点击 | 点击后清空条件 |

### 3.3 列表操作区

| 操作项 | 组件类型 | 初始状态 | 约束规则 | 显示位置 |
| --- | --- | --- | --- | --- |
| 新增 | 按钮 | 可用 | 满足权限可点击 | 居右 |
| 删除 | 按钮 | 置灰 | 勾选用户后激活 | 居右 |

## 四、交互说明

| 交互类型 | 交互可用性 | 正向交互 | 异常交互 |
| --- | --- | --- | --- |
| 新增 | 满足权限可用 | 打开新增用户弹窗 | 无权限时按钮不展示 |
| 查询 | 始终可用 | 按条件筛选用户列表 | 条件非法时提示错误 |
| 重置 | 始终可用 | 清空筛选项并恢复默认列表 | 无 |
| 查看 | 始终可用 | 打开详情弹窗查看用户信息 | 无 |
| 编辑 | 满足权限可用 | 打开编辑弹窗并反显当前数据 | 被锁定账号不可编辑 |
| 删除 | 仅系统管理员可用 | 二次确认后删除用户 | 超级管理员账号不可删除 |
| 导出 | 满足权限可用 | 下载 Excel 导出文件 | 无数据时提示暂无数据 |

## 十一、特殊说明

### 11.1 编码规则

| 项目 | 规则说明 |
| --- | --- |
| 用户编码 | USER + 4 位序号 | 例如 USER0001 |
| 唯一性 | 全局唯一 | 不允许重复 |
| 生成方式 | 系统自动生成 | 不允许手工修改 |

### 11.2 删除及修改限制

超级管理员账号不可删除；被系统内置引用的账号不可删除；被锁定账号需先解锁后再编辑。

### 11.3 其他特殊规则

- 手机号在列表页展示时需按脱敏规则显示。
- 账号状态为"禁用"时不可登录系统。
